# Video Game Market Analysis: Identifying Success Patterns (2016)

## Project Overview
This project was developed for the online store **Ice**, a global video game retailer. The goal was to analyze historical data from open sources—including user/expert reviews, genres, platforms, and sales—to identify patterns that determine whether a game will be successful. 

The analysis focuses on planning a marketing campaign for 2017 based on data up to December 2016.

## Key Objectives
* **Data Preprocessing:** Cleaning missing values, handling "TBD" (To Be Determined) entries, and optimizing data types for analysis.
* **Exploratory Data Analysis (EDA):** Identifying the life cycle of gaming platforms and determining the relevant period for forecasting.
* **Regional Profiling:** Creating specific user profiles for North America (NA), Europe (EU), and Japan (JP) based on genre and platform preferences.
* **Statistical Hypothesis Testing:** Using T-tests to compare user ratings between platforms (Xbox One vs. PC) and genres (Action vs. Sports).

## Technical Skills Demonstrated
* **Data Manipulation:** Python, Pandas, NumPy.
* **Data Visualization:** Matplotlib, Seaborn.
* **Statistical Analysis:** Hypothesis testing (SciPy), ESRB rating impact analysis, and correlation studies (Critic vs. User scores).

## Key Insights
* Identified the "Top 5" platforms per region and how market share shifts geographically.
* Determined the impact of ESRB ratings on regional sales.
* Analyzed the correlation between expert reviews and global sales performance.

## How to Run
1. Clone this repository.
2. Ensure you have the `games.csv` file in the same directory.
3. Run the `Video_Game_Sales_Analysis.ipynb` notebook in a Jupyter environment.
